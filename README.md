# 👋 Personal Introduction Program

## Project Overview

The Personal Introduction Program is a beginner-friendly Python application that collects a user's personal information, including their name, age, and favorite hobby. It then displays a personalized introduction using formatted output.

This project demonstrates the fundamentals of Python programming, including user input, variables, string formatting, and output statements.

### Objectives

- Learn how to take user input.
- Store data in variables.
- Display formatted output using f-strings.
- Understand basic Python syntax.
- Practice writing clean and readable code.

- ## Setup Instructions

### Requirements

- Python 3.14
- VS Code (Recommended)

### Installation

1. Install Python from https://python.org
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the program using:

```bash
